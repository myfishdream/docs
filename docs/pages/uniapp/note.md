# uniapp

## 组件

在项目文件夹中创建一个与`pages`同级的文件夹`components`用于存放子组件

**注意：** 在HBuilder X中如果文件夹名字创建正确右键文件夹会有**新建组件**选项

**最后在有需要的地方直接使用即可，在uniapp不用引入可以直接使用**（<u>创建子组件时候要勾选**创建同名目录**</u>）
